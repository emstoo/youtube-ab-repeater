# YouTube AB Repeater

A lightweight, dark-themed single-page application (SPA) to repeat specific sections of YouTube videos.

[Demo site](https://emstoo.github.io/youtube-ab-repeater/)

## Features

- **AB Repeat**: Set start (A) and end (B) points during playback or via manual input.
- **URL Synchronization**: Video ID and loop points are automatically saved in the URL query string.
- **Bookmarkable**: Simply bookmark the URL to save and return to your specific loop settings.
- **Zero Dependencies**: No frameworks or external CSS libraries used (only the official YouTube IFrame API).
- **Responsive Dark Mode**: Optimized for both PC and mobile with a sleek dark interface.
